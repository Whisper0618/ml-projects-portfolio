Machine Learning Projects Portfolio

This page showcases multiple machine learning projects developed using Python, Jupyter Notebook, and scikit-learn.

1. Classification

A supervised learning pipeline applying classification algorithms to structured datasets. Models used include: K-Nearest Neighbors, Decision Tree, Random Forest, and Support Vector Machine (SVM). The pipeline covers data preprocessing, train-test split, model comparison using accuracy, confusion matrix, and AUC.

2. Clustering
